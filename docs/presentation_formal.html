Модель машинного обучения

Выбранный алгоритм: XGBoost (Extreme Gradient Boosting)

Ансамблевый метод, основанный на градиентном бустинге деревьев решений

Преимущества XGBoost:

  • Высокая точность классификации
  • Устойчивость к переобучению
  • Эффективная работа с разреженными данными
  • Встроенная регуляризация

Параметры модели:

  • n_estimators = 100
  • max_depth = 6
  • learning_rate = 0.1

Функционал мобильного приложения

  • Ручной ввод URL — проверка ссылки через текстовое поле
  • QR-сканер — сканирование QR-кодов с URL
  • История проверок — сохранение результатов
  • AI-консультант — чат-бот на базе Google Gemini
  • Объяснение решений (XAI) — почему URL классифицирован как фишинг

Особенности:

  • Поддержка iOS и Android
  • Whitelist казахстанских доменов
  • Обнаружение typosquatting (kaspii.kz, kasp1.kz)
